Goals Tab Direction

Pick the product direction for the new main tab named Goals.

A

Auto-computed planning dashboard

Goals like FIRE, emergency fund, debt payoff, and savings rate are calculated from accounts, balances, transactions, budgets, and reports data.

B

Manual goal tracker

Users create goals, enter target amounts and dates, and update or map progress manually.

C

Hybrid, auto-first

Start with smart default goals computed from existing app data, then let users customize assumptions and add their own goals.

My recommendation is C: hybrid, auto-first. It makes the tab useful immediately while leaving room for user-specific planning.
